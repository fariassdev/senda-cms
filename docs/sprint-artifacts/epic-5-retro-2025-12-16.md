## Challenges and Learnings

### Challenge: File Download Filenames

**Issue:** S3/Storage pre-signed URLs often ignore the `download` attribute or force the stored filename (internal ID) instead of a user-friendly name.
**Solution:** Implemented a client-side `fetch` -> `blob` -> `URL.createObjectURL` pattern.
**Learning:** For user-facing asset downloads, do not rely on raw 3rd-party URLs if you need custom naming. Handle the byte stream client-side when possible (for reasonable file sizes).

### Challenge: React Hydration & Audio

**Issue:** Initial implementation of audio player caused hydration mismatches due to `window` object access.
**Solution:** Proper use of `useEffect` and `Mounted` checks to ensure audio logic only runs on the client.
**Learning:** Media elements tied to global state require careful lifecycle management in Next.js 15.
